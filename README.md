# ES3 Game Assistance

一个本地运行的 ES3 游戏存档修改器。项目受 savefile.space 的交互方式启发：用户选择本地 `.es3` 存档，页面解析出可编辑字段，修改后下载新的存档文件替换原文件。

当前版本以《吾今有世家》的 `GameData.es3` 为主要样例，内置了该游戏的物品、书籍、角色属性等补充资料，用于把存档里的 ID 字段尽量映射为可读语义。

## 特性

- 浏览器本地解析 `.es3`、`.json`、纯文本 JSON 存档。
- 不上传存档，不依赖后端服务。
- 支持快速编辑、树形字段浏览、原始 JSON 查看与应用。
- 支持《吾今有世家》模板识别与字段语义补全。
- 支持资源、家族基础数值、成员、物品、书籍等常用字段编辑。
- 支持物品搜索、分类筛选、批量数量修改、添加未拥有物品、移除已有物品。
- 修改完成后下载新存档，原始文件不会被页面直接覆盖。

## 使用方式

1. 打开 `index.html`。
2. 点击“选择 .es3”或将存档拖入页面。
3. 在快速编辑、树形视图或原始 JSON 中修改字段。
4. 点击下载按钮生成修改后的存档。
5. 备份原存档后，再用新文件替换游戏存档。

## 文件结构

```text
.
├── index.html                         # 主页面与存档编辑器逻辑
├── templates/
│   ├── wjysj-template.js              # 《吾今有世家》字段模板
│   └── wjysj-reference-data.js        # 物品、书籍、属性等参考数据
```

## 当前支持范围

项目的解析核心面向 JSON 结构的 ES3 存档，具备一定通用性；但字段语义、ID 映射和快速编辑表单目前主要围绕《吾今有世家》完善。

对于其他游戏的 ES3 存档，如果文件内容也是可读 JSON，可以先通过树形视图和原始 JSON 进行查看与修改。要获得完整的中文字段名、枚举说明和专用表单，需要继续补充对应游戏的模板与参考数据。

## 安全提示

- 修改存档前务必备份原文件。
- 不建议一次性写入过大的数值，部分游戏会校验数值范围或触发异常。
- 下载的新存档建议先在测试档中验证，再替换长期游玩的正式存档。

## 数据说明

仓库中的《吾今有世家》相关资料仅用于本地存档解析和字段说明。不同游戏版本可能会调整字段结构或 ID 含义，如发现字段不匹配，应以当前游戏实际存档为准。
